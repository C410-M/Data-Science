Olá pessoal!

Neste código estou lendo a lista das 500 maiores empresas do Brasil, organizadas pelo Site:

https://www.econodata.com.br/maiores-empresas/todo-brasil

Como o Site possui a tecnologia "Lazy Loading", fez-se necessário a implementação de algumas funções que:

- Clica no botão: Expandir Lista
- Percorra toda a lista, com um time para carregar os dados
- Colhe os dados da lista

No arquivo App.py crio um Dashboard que filtra dados comparativos entre as 500, 250 e 50 maiores empresas;
Também é possível filtrar essa empresas por faturamento e estado;
Por fim, no último gráfico, fique a vontade para, comparar dados de faturamento e setor entre 2 dessas empresas.

PS: Apesar de ter coletado os dados do Site Econodata, eles são públicos e podem ser achados gratuitamente em outras fontes da internet.
